Bons Plans
La communauté des amis pour partager et découvrir les meilleurs spots autour de soi.
Aperçu
Application 100 % web (1 fichier HTML) permettant de :
parcourir des posts classés par thèmes (Restaurants, Activités, Spectacles…)
créer, modifier, supprimer un post avec photo et note 1-5 ⭐
partager un lien pré-formaté vers un post
Front : vanilla JS – Back : Supabase (PostgreSQL + API REST)
Stack technique
HTML / CSS / ES6 modules (aucun build)
Supabase (tables themes, posts, post_images, comments, replies, user_profiles, invitations)
Web Share API + Clipboard pour le partage
